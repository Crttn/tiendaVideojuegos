Instrucciones para el uso del programa:

	1) Agregar usuario, se pedirán los parámetros (Dni, Nombre, Apellidos, Correo).

	2) Eliminar usuario, se introduce el Dni del usuario a eliminar.

	3) Buscar usuario, se introduce el Dni del usuario a buscar.

	4) Salir del programa.
